EWS for Office 365 Release Notes - Published by Splunk April 04, 2022

Version 2.10.0 - Released April 04, 2022
  • Added support for ingestion of empty attachments [PAPP-24108]
  • Added logging statement when ingesting attachments with Content-Disposition as None [PAPP-17687]
  • Updated documentation related to API permissions [PAPP-24544]
Version 2.9.0 - Released February 15, 2022
  • Added validation for the extraction of URL artifacts [PAPP-23436]
  • Fixed a bug for the extraction of the URL value from the src attribute
Version 2.8.12 - Released February 07, 2022
  • Added support for Python 3.9
Version 2.8.10 - Released January 25, 2022
  • Added a new action 'trace email' [PAPP-21222]
  • Fixed a bug in 'get email' and 'run query' actions where all the subfolders weren't being populated [PAPP-16851]
  • Changed the hashing algorithm to SHA256 when running in FIPS mode [PAPP-20099]
  • Bug fixes in the 'get email' action [PAPP-10215]
  • Added ability to ingest the actual email as a vault item [PAPP-17141]
  • Added Preprocess script in app documentation [PAPP-15998]
  • Improved performance of scheduled polling for the 'oldest first' ingestion [PAPP-19684]
